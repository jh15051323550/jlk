# Hotel Inventory Management System

This project is designed to manage the inventory for a hotel, allowing for the tracking and management of rooms, bookings, and guest information.

## Features
- Add, update, and delete rooms
- Manage bookings
- Track availability
- Generate reports

## Getting Started

### Prerequisites
- Python 3.x
- Required packages are listed in requirements.txt

### Installation
1. Clone the repository
2. Install the requirements using `pip install -r requirements.txt`
